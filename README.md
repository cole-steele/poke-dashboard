# Poke Dashboard

A Pokémon dashboard built with Next.js, Supabase, and SCSS.

## Stack

- **Next.js** — App Router, server-side rendering, API proxy
- **Supabase** — Postgres database & auth
- **SCSS** — Modular styles with variables and mixins

## Getting Started

Copy the env template and fill in your Supabase project credentials:

```bash
cp .env.local.example .env.local
```

Then run the dev server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.
